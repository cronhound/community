# Examples

Real-world cronhound setups, contributed by the community. Each one is a working
recipe you can adapt for your own use.

## Available examples

- [Slack integration](./slack-integration.md) — Route cronhound alerts to a Slack channel

## Contributing an example

Got a setup that works well? Help other users learn from it.

1. Fork this repo
2. Add a new `.md` file in this folder (use `slack-integration.md` as a template)
3. Include: the problem you were solving, your setup, the cronhound config, and any gotchas
4. Add a link to it in the list above
5. Open a PR

Good examples are:
- **Specific** — real use case, not a hypothetical
- **Complete** — someone should be able to follow along and get the same result
- **Honest** — include what didn't work, not just the happy path

Questions about contributing? Ask in
[Q&A](https://github.com/cronhound/community/discussions/categories/q-a).
